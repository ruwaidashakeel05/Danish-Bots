# Learnobots

Learnobots is a simple web and Electron-based learning project.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm start
   ```

## Publish to GitHub

1. Create a new repository on GitHub.
2. In the project folder, run:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
   git push -u origin main
   ```

If you want, I can help you choose the exact GitHub repository name and the final push commands.
